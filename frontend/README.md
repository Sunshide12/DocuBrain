# Frontend — DocuBrain

> **Esta carpeta se construye en la Fase 7.**

El frontend será una **SPA con Vue 3 + Vuetify 3 + Apollo Client** que consume el API GraphQL de Laravel.

## Stack planeado

- **Vue 3** — Composition API
- **Vuetify 3** — UI components (Material Design)
- **Apollo Client** — GraphQL queries, mutations y subscriptions
- **Vite** — bundler

## Por qué arranca en Fase 7

La spec de DocuBrain sigue un enfoque **API-first**: el backend GraphQL se desarrolla y prueba completamente (Fases 0–6) antes de construir la UI. Esto garantiza que el API sea sólido, testeado y maduro antes de depender de él en el frontend.

Durante las fases 0–6, el API se testea con **GraphQL Playground** en `http://localhost:8000/graphql-playground`.
